# تحليل البيانات الاجتماعية: تأثير الإعلام الأجنبي على القيم 📊

### نظرة عامة على المشروع
دراسة ميدانية وكمية تهدف إلى قياس وتحليل تأثير الأفلام والمسلسلات الأجنبية على القيم الاجتماعية. يبرز هذا المشروع القدرة على دمج المفاهيم في **علم الاجتماع** مع مهارات **تحليل البيانات** لاستخراج رؤى عميقة حول السلوك البشري والتوجهات.

### الأدوات والمهارات المستخدمة
- **البرنامج الإحصائي:** SPSS.
- **المهارات التحليلية:** تصميم الاستبيانات، التحليل الوصفي، اختبار الفرضيات، والتحليل الكمي.
- **جمع وتنظيف البيانات:** تفريغ البيانات الخام ومعالجتها إحصائياً.

### الخطوات المتخذة
1. **تصميم أداة الدراسة:** بناء استبيان دقيق ومقنن لجمع البيانات الميدانية.
2. **المعالجة والتنظيف:** إدخال البيانات في برنامج SPSS، معالجة الإجابات المفقودة، وترميز المتغيرات.
3. **التحليل الإحصائي:** إجراء التحليلات الوصفية لاستكشاف التوجهات، واختبار الفرضيات الإحصائية للإجابة على تساؤلات الدراسة.
4. **استخراج الرؤى (Insights):** تحويل الأرقام الإحصائية إلى نتائج ملموسة وتوصيات واضحة حول السلوك الاجتماعي.

### القيمة المضافة
يثبت هذا المشروع القدرة المتقدمة على التعامل مع "البيانات البشرية" المعقدة، وتقديم تحليلات تتجاوز الأرقام لتشمل فهم الدوافع والسلوكيات، وهي مهارة استراتيجية مطلوبة بشدة في مجالات استشارات الأعمال وتحليل سلوك العملاء.
